# Zyphos Language Design Pressure

Zyphos pressures Weksa toward languages where memory, ecology, reproduction,
movement, and negotiated boundary are not background. They are the political
substrate.

The design target is not alien flavor pasted onto Earth institutions. Every
major linguistic distinction should be tested against the setting rule:

> Does this institution or name reveal an ecological deal, reproductive pressure,
> memory technology, or path-dependent consequence?

## Ontology Pillars

- mutable memory endosymbiosis
- sentience across scale
- ecology as infrastructure
- memory as obligation, leverage, and negotiated access
- organisms, routes, forests, reefs, and landscapes as possible political actors
- boundary permeability as foreign policy
- reproductive sites as commons or strategic terrain
- path dependency as a civilizational constraint
- disconnection as survival, refusal, or wound

## Airawa Pressures

Airawa language work must account for:

- mother-tree mediated communal memory
- archival dependency
- ritual labor and nutrient obligation
- imperial biological-memetic infrastructure
- caste and eugenic pressure
- disconnected tree networks that survived by severing themselves from saturated
  substrate
- imperial labels versus native nation-scale interbeing names

The Airawa Empire should not be named like a normal state with wet nouns glued
on. It is an artificial quarter-continent clausiform exception imposed through
biological-memetic control.

### Provisional Airawa Phonology Notes

Airawa phonology is not finalized, but the current design pressure distinguishes
it from Sa'auei'a softness. Airawa names should feel sharper, more dorsal and
coronal, and more willing to close around hard edges.

Provisional tendencies:

- dorsal and coronal plosives should be prominent: `k`, `g`, `q`, `t`, `d`;
- sibilant and postalveolar fricatives should carry social texture: `s`, `z`,
  `sh` / `zh`;
- `r` may be trilled in some Airawa registers;
- syllables can tolerate clipped or closed shapes where Sa'auei'a forms would
  flow longer and wetter;
- avoid making native Airawa names read like smooth Sa'auei'a route names unless
  the form is explicitly a loan, exonym, or cross-continental register.

Current scratch name: **Druzkai**, with trilled `r` and `z` pronounced as a
voiced postalveolar fricative /zh/ [ʒ]. The name is being considered for a
Zyphos Face drawn from a non-imperial Airawa archival/fungal-road tradition: an
archivist shaped by memory as kinship, not content.

## Sa'auei'a Pressures

Sa'auei'a language work must account for:

- advanced nomadic civilization without permanent urban dominance
- squad-sized family units
- sacred communal breeding grounds
- remembered routes
- portable institutions
- ecological reputation
- reciprocity routes and seasonal corridors
- camps, commons, and partner sites as distinct name targets

The Sa'auei'a continent should not read as empty wilderness. Its language should
have ways to name civilization that moves, recurs, bargains, and is remembered by
ecology.
