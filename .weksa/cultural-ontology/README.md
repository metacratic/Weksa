# Cultural Ontology CultCache Seeds

This directory stores Weksa cultural ontology as CultCache `.cc` witnesses.

## Starter Database

- `starter-targets.cc`

This store contains one catalog entry and three draft guarded profile entries:

- `weksa.cultural_ontology.pt-BR.vanilla_brazilian.v0`
- `weksa.cultural_ontology.en.transatlantic.v0`
- `weksa.cultural_ontology.ja-JP.kanto.contemporary.v0`

The catalog entry is:

- `weksa.cultural_ontology.catalog.seed_targets.v0`

The `.cc` file is the typed state witness. This README is only an inspection
guide; it does not own cultural ontology truth.

The profiles follow `schemas/cultural-ontology-profile.md`: salience axes and
linguistic affordances are nested typed state with feature IDs, activation
gates, constraints, indexicality, evidence, uncertainty, and trace labels.

Qdrant retrieval may later propose evidence bundles for these profiles. It does
not own accepted profile state.
